Política de Privacidade

1. Coleta de Dados

O ProtetiCalc armazena apenas informações necessárias para funcionamento do sistema, como cadastro e histórico de cálculos. Nenhum dado sensível ou financeiro é coletado.

2. Privacidade e Segurança

Os dados fornecidos são utilizados exclusivamente para o funcionamento do aplicativo. Não há integração com APIs ou serviços de terceiros.

3. Direitos do Usuário

Todos os dados são pessoais, intransferíveis e armazenados de maneira segura. O ProtetiCalc não compartilha informações com terceiros sob nenhuma circunstância.

4. Compartilhando informação

O Usuário quando logado, apenas tem a possibilidade de compartilhar determinado dado do histórico salvo com terceiro, enviando um determinado cálculo, caso queira.

5. Alterações na Política

Os usuários têm o direito de solicitar a exclusão de seus dados armazenados no aplicativo. Para isso, basta entrar em contato com o suporte.

6. Coleta de Dados

O criador reserva-se o direito de alterar esta política, notificando os usuários sobre mudanças relevantes.

Se tiver dúvidas sobre esta política, entre em contato pelo e-mail: proteticalc@gmail.com.
